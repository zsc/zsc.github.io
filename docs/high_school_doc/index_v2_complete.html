核心概念

受力分析是根据物体的运动状态和周围环境，分析物体受到的所有力的过程。它是解决力学问题的第一步，也是最关键的一步。

✨

建模思想

把复杂的实际问题简化为物理模型，是物理学的核心方法。受力分析的本质就是建立力学模型。

  2. 分析物体状态 - 静止？匀速？加速？
  3. 寻找施力物体 - 每个力都要找到"谁给的"

1.2 力的分类体系

分类依据 类型 特点 常见例子
按性质分 重力 始终竖直向下 G = mg
弹力 垂直于接触面 压力、支持力、拉力
摩擦力 平行于接触面 静摩擦、滑动摩擦
场力 不需要接触 电场力、磁场力
按接触分 接触力 需要直接接触 弹力、摩擦力
非接触力 超距作用 重力、电磁力
⚠️

特别提醒

避免使用"主动力"、"被动力"这样不科学的分类。应该根据力的性质和产生条件来分类。

🤔 思考题：一个静止在水平桌面上的木块，受到哪些力？每个力的施力物体是什么？

答案：

  1. 重力 G：施力物体是地球，方向竖直向下
  2. 支持力 N：施力物体是桌面，方向竖直向上

因为木块静止，所以 G = N，二力平衡。

2.1 正交分解法——"化整为零"的艺术

⚠️

重要提醒

核心思想

将一个矢量（通常是力或加速度）分解到两个相互垂直的坐标轴上，把复杂的二维平面问题转化为两个独立的一维直线问题。

x轴方向：F合x = max
y轴方向：F合y = may

如何选择坐标轴？【核心技巧】

📋

示例

坐标轴选择的黄金原则
原则 适用情况 优势
原则一（首选）
沿加速度方向建轴		 物体有明确的加速度方向 一个方向变为平衡方程
F合⊥ = 0
原则二（备选）
让更多力落在轴上		 静止或匀速直线运动
（a = 0）		 减少力的分解
降低计算复杂度

口诀：有加速度，就沿加速度方向建轴；无加速度，就让尽量多的力免于分解。

📋

示例

典型例题：斜面上的加速物块

题目：质量m = 2kg的物块，在与水平方向成θ = 37°角的恒力F = 20N作用下，沿粗糙水平地面向右做匀加速直线运动。物块与地面间的动摩擦因数μ = 0.5。求物块的加速度a和地面对物块的支持力N。

（g = 10 m/s²，sin37° = 0.6，cos37° = 0.8）

物理示意图

解题步骤：
  1. 受力分析
    • 重力 G = mg = 20N（竖直向下）
    • 支持力 N（竖直向上）
    • 拉力 F = 20N（与水平成37°角）
    • 滑动摩擦力 f = μN（水平向左）
  2. 选择坐标轴

    加速度方向明确：水平向右。根据原则一，建立水平x轴（向右为正）和竖直y轴（向上为正）。

  3. 正交分解
    • Fx = Fcos37° = 20 × 0.8 = 16N
    • Fy = Fsin37° = 20 × 0.6 = 12N
  4. 列方程求解
    y方向（平衡）：N + Fy - mg = 0
    N + 12 - 20 = 0
    N = 8N
    x方向（加速）：Fx - f = ma
    其中 f = μN = 0.5 × 8 = 4N
    16 - 4 = 2a
    a = 6 m/s²

2.2 整体法与隔离法——"宏观与微观"的切换

📋

示例

方法对比
方法 核心思想 优点 缺点
整体法 将系统看作一个整体
忽略内力		 求系统加速度最快
方程简单		 无法求内力
隔离法 分析单个物体
考虑所有作用力		 可以求内力 方程较多
计算复杂
⚠️

重要提醒

使用策略【黄金法则】
  1. 判断能否用整体法

    唯一条件：系统内所有物体的加速度大小和方向必须完全相同

  2. 根据所求选择方法
    • 只求系统加速度 → 优先用整体法
    • 要求内力（拉力、压力等） → 必须用隔离法
    • 既求加速度又求内力 → 先整体后隔离
📋

示例

典型例题：连接体问题

题目：光滑水平面上，质量mA = 2kg和mB = 3kg的两物体用轻绳连接。用水平力F = 10N拉物体B，求：

  1. 系统的加速度a
  2. 绳子的拉力T

物理示意图

解题过程：
✨

技巧提示

(1) 求系统加速度a——使用整体法

  • A和B加速度相同，可用整体法
  • 整体质量：M = mA + mB = 5kg
  • 整体受力：只有外力F = 10N
  • 由F = Ma得：a = F/M = 10/5 = 2 m/s²
✨

技巧提示

(2) 求绳子拉力T——使用隔离法

隔离物体A（受力较简单）：

  • A只受绳子拉力T（水平向右）
  • 对A应用牛顿第二定律：T = mA·a
  • T = 2 × 2 = 4N

验证：隔离物体B

  • B受拉力F（向右）和绳子拉力T（向左）
  • F - T = mB·a
  • 10 - T = 3 × 2
  • T = 4N ✓
思考题

三个相同的物块A、B、C叠放在光滑水平面上，用水平力F拉最下面的C。问：

  1. 能否用整体法求系统加速度？
  2. 如何求A和B之间的摩擦力？
  3. 如果C与地面有摩擦，分析方法有何改变？

1. 可以，因为三个物体一起运动，加速度相同

2. 隔离A，A受到的摩擦力提供其加速度

3. 仍可用整体法，但要考虑地面摩擦力这个外力

3.1 临界与极值问题——"恰好"状态的数学表达

⚠️

重要提醒

核心思想

临界问题的本质是分析一个物理过程的转折点。当某个物理量变化时，物体的运动状态也随之变化。临界状态就是这两种状态的分界点。

关键词识别："刚好"、"恰好"、"至多"、"至少"、"最大"、"最小"

临界条件总结

📋

示例

临界状态 临界条件 数学表达
即将滑动 静摩擦力达到最大值 fs = μsN
即将脱离接触 接触面弹力为零 N = 0
绳子即将断裂 拉力达到最大值 T = Tmax
圆周运动最高点 仅重力提供向心力 mg = mv²/r
📋

示例

例题1：斜面上物体的临界滑动

题目：质量m = 2kg的物体放在倾角θ = 37°的斜面上，静摩擦因数μs = 0.5。用平行于斜面向上的力F拉物体，求：

  1. 使物体保持静止，F的取值范围
  2. 若F = 25N，物体的加速度（动摩擦因数μk = 0.3）

（g = 10m/s²，sin37° = 0.6，cos37° = 0.8）

向上 f向下 37°

物理示意图

解答：

(1) 静止时F的范围

  • 垂直斜面：N = mgcosθ = 2×10×0.8 = 16N
  • 最大静摩擦力：fmax = μsN = 0.5×16 = 8N
  • 重力沿斜面分量：mgsinθ = 2×10×0.6 = 12N
✨

技巧提示

临界状态一：即将下滑（F最小）

摩擦力沿斜面向上，达到最大值

Fmin + fmax = mgsinθ
Fmin + 8 = 12
Fmin = 4N
✨

技巧提示

临界状态二：即将上滑（F最大）

摩擦力沿斜面向下，达到最大值

Fmax = mgsinθ + fmax
Fmax = 12 + 8
Fmax = 20N

答案：4N ≤ F ≤ 20N

(2) F = 25N时的加速度

F > Fmax，物体向上滑动，受滑动摩擦力

fk = μkN = 0.3×16 = 4.8N
F - mgsinθ - fk = ma
25 - 12 - 4.8 = 2a
a = 4.1 m/s²（沿斜面向上）

3.2 瞬时问题——抓住"不变"与"突变"

⚠️

重要提醒

瞬时问题的黄金法则
力的类型 瞬时变化特性 原因
弹簧弹力 不突变 形变量不能瞬时改变
绳子拉力 可突变 约束条件改变
接触面弹力 可突变 接触状态改变
摩擦力 可突变 随正压力突变
✨

技巧提示

解题三步曲
  1. 分析变化前：求出各力大小，特别是弹簧弹力
  2. 分析变化后瞬间：哪些力消失/突变，哪些力保持不变
  3. 应用牛顿第二定律：求瞬时加速度
📋

示例

例题2：竖直悬挂系统的瞬时问题

题目：两个质量分别为mA = 1kg和mB = 2kg的物块，用轻质弹簧连接后用细绳悬挂，系统静止。剪断细绳瞬间，求A、B的加速度。

物理示意图

解答：

第一步：分析剪断前（平衡态）

  • 对B：F弹簧 = mBg = 2×10 = 20N（向上）
  • 对整体：T = (mA + mB)g = 3×10 = 30N

第二步：剪断瞬间的受力变化

  • 绳子拉力T：30N → 0（突变）
  • 弹簧弹力F弹簧：20N（保持不变）

第三步：求瞬时加速度

对A：F合A = mAg + F弹簧 = 10 + 20 = 30N（向下）
aA = F合A/mA = 30/1 = 30 m/s²（向下）

对B：F合B = mBg - F弹簧 = 20 - 20 = 0
aB = 0
📋

示例

对比：剪断弹簧的情况

若剪断的是弹簧而非细绳，结果完全不同：

  • 弹簧被剪断，弹力立即消失（弹簧不存在了）
  • A：仍被绳拉住，aA = 0
  • B：自由落体，aB = g = 10 m/s²
思考题

如图，小车在光滑水平面上，车厢内用细绳和弹簧悬挂两个小球。剪断细绳瞬间：

  1. 左边小球的加速度是多少？
  2. 右边小球的加速度是多少？
  3. 小车会向哪边加速？
  1. 绳被剪断，拉力消失，左球只受重力，a = g（向下）
  2. 弹簧弹力不突变，若原来平衡，则a = 0
  3. 系统水平方向动量守恒，左球向下运动，小车不会水平加速
瞬时问题分析流程

剪断前平衡分析
↓
弹簧弹力 = ?（记住此值）
↓
剪断后受力分析
↓
弹簧弹力不变 + 其他力突变
↓
F = ma 求加速度

4.1 弹簧的核心特性再深化

⚠️

重要提醒

胡克定律的本质理解
F = kx
  • F：弹簧的弹力（N）
  • k：弹簧的劲度系数（N/m），反映弹簧的"软硬"程度
  • x：弹簧的形变量（m），不是弹簧长度！

记住：x = |L - L₀|，其中L₀是弹簧原长

✨

技巧提示

弹簧的瞬时特性（再次强调）

弹簧两端连接物体的加速度可以突变
但弹簧自身的弹力（由形变量决定）不能突变

4.2 弹簧的串并联

📋

示例

串联弹簧系统
eq F

物理示意图

串联特点：
  • 受力特点：各弹簧受力相等，F₁ = F₂ = F
  • 形变特点：总形变量等于各弹簧形变量之和，xtotal = x₁ + x₂
等效劲度系数推导：
由 F = k₁x₁ 得：x₁ = F/k₁
由 F = k₂x₂ 得：x₂ = F/k₂
总形变：xtotal = x₁ + x₂ = F/k₁ + F/k₂ = F(1/k₁ + 1/k₂)
等效弹簧：F = keq·xtotal
因此：1/keq = 1/k₁ + 1/k₂

结论：串联后系统变"软"，keq < min(k₁, k₂)

📋

示例

并联弹簧系统
eq

物理示意图

并联特点：
  • 形变特点：各弹簧形变量相等，x₁ = x₂ = x
  • 受力特点：总拉力等于各弹簧弹力之和，Ftotal = F₁ + F₂
等效劲度系数推导：
F₁ = k₁x，F₂ = k₂x
总拉力：Ftotal = F₁ + F₂ = k₁x + k₂x = (k₁ + k₂)x
等效弹簧：Ftotal = keq·x
因此：keq = k₁ + k₂

结论：并联后系统变"硬"，keq > max(k₁, k₂)

4.3 弹簧振子模型

⚠️

重要提醒

简谐运动的条件

回复力F与位移x成正比，且方向相反：F = -kx

加速度：a = F/m = -(k/m)x

这是简谐运动的动力学特征

📋

示例

水平弹簧振子

物理示意图

周期公式：
T = 2π√(m/k)

注意：周期只与m和k有关，与振幅A无关！

4.4 弹簧与复杂情景的组合

📋

示例

例题：弹簧-斜面系统

质量m = 1kg的物块通过劲度系数k = 100N/m的弹簧连接在光滑斜面底部。斜面倾角θ = 30°。将物块从弹簧原长位置静止释放，求：

  1. 物块的平衡位置
  2. 物块的最大速度
  3. 振动的振幅

物理示意图

解答：

(1) 平衡位置

在平衡位置，弹簧弹力等于重力沿斜面的分量：

kx₀ = mgsinθ
x₀ = mgsinθ/k = 1×10×0.5/100 = 0.05m = 5cm

(2) 最大速度

最大速度出现在平衡位置。从原长到平衡位置，由能量守恒：

mgx₀sinθ = ½kx₀² + ½mvmax²
vmax = x₀√(g·sinθ/x₀ - k/2m)
vmax = 0.05√(100 - 50) = 0.35 m/s

(3) 振幅

从原长释放，振幅A = x₀ = 5cm

挑战题

两个相同的弹簧（k = 200N/m）和两个相同的物块（m = 0.5kg）按下图连接。整个系统竖直悬挂，求：

  1. 两弹簧的伸长量
  2. 剪断连接两物块的绳子瞬间，各物块的加速度

(1) 伸长量：

  • 下弹簧：只承受B的重力，Δx₂ = mg/k = 0.5×10/200 = 0.025m = 2.5cm
  • 上弹簧：承受A和B的总重力，Δx₁ = 2mg/k = 2×0.5×10/200 = 0.05m = 5cm

(2) 剪断瞬间：

  • 弹簧弹力不突变！
  • A：F = mg + kΔx₂ = 5 + 5 = 10N，aA = 20m/s²（向下）
  • B：F = mg - kΔx₂ = 5 - 5 = 0，aB = 0
