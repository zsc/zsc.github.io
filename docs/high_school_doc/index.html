第一章：基础回顾与思想升华（地基篇）

目标

纠正不良习惯，建立"程序化"分析思维，确保基础分不失。

1.1 受力分析的"金科玉律"

力的科学分类方法

为了系统地进行受力分析，我们需要对力进行科学的分类。这里介绍两种并行使用的分类方法：

按力的性质分类

这是最基本、最科学的分类方法，帮助我们理解每种力的来源和特点：

  • 重力 (G)：地球对物体的吸引力，G = mg，方向永远竖直向下
  • 弹力 (N)：物体发生弹性形变时产生的力，包括支持力、压力、拉力等
  • 摩擦力 (f)：两个接触面间阻碍相对运动或相对运动趋势的力
  • 电场力 (Fe)：带电体在电场中受到的力，Fe = qE
  • 磁场力 (FB)：运动电荷或通电导线在磁场中受到的力
按是否接触分类

这种分类方法有助于建立清晰的分析步骤：

类型 特点 常见力 分析要点
非接触力（场力） 不需要接触即可产生 重力、电场力、磁场力 只要在场中就存在
接触力 必须接触才能产生 弹力、摩擦力 检查所有接触点
分析顺序口诀

"先画重力，再找接触；有接触，就可能有弹力和摩擦力。"

这个顺序既科学又实用，能有效避免遗漏力的情况。

力的矢量性

核心概念：力是矢量，既有大小又有方向。画受力图的过程，本质上就是在确定作用在物体上的所有矢量。

在受力分析中，我们必须同时关注：

  • 力的大小（用箭头长短表示）
  • 力的方向（用箭头指向表示）
  • 力的作用点（虽然常简化到质心）

受力图（FBD）的"洁癖"原则

黄金三步法 + 一个核心原则

核心原则：隔离物体（Isolate the Body）

在分析任何一个物体的受力时，必须先把它从周围的环境中"隔离"出来，只分析其他物体对它的力。

画受力图的完整步骤
  1. 明确对象，画隔离体

    选定研究对象，想象把它从周围环境中"抠"出来，单独分析。

  2. 先画场力，再找接触
    • 画出重力 G（竖直向下，作用在重心）
    • 检查是否有其他场力（电场力、磁场力）
    • 沿对象边界"走一圈"，检查所有接触点/面
  3. 分析接触，判断弹摩

    对每个接触点：

    • 判断弹力：有挤压？→ 有弹力N，垂直于接触面指向被研究对象
    • 判断摩擦力：
      • 三个条件：①接触面粗糙 ②有正压力 ③有相对运动（趋势）
      • 方向：与相对运动（趋势）方向相反
  4. 检查其他已知力

    如绳子的拉力T、人为施加的推力F等

  5. 规范标注与最终校验
    • 为每个力标上标准符号（G、N、f、T、F等）
    • 检查：每个力都能找到施力物体吗？
    • 提醒：只画"别人给我的力"，不画"我给别人的力"
    • 禁忌：不画分力、合力、"惯性力"、"向心力"
牛顿第三定律的提醒

记住：你画的每一个力，都有一个等大反向的反作用力作用在施力物体上。虽然在受力图中不画反作用力，但这个意识对理解力的本质和解决连接体问题至关重要。

作用力与反作用力 vs 平衡力：

  • 作用力与反作用力：作用在不同物体上，性质相同
  • 平衡力：作用在同一物体上，合力为零

1.2 弹力与摩擦力的再认识

例子1：静止在水平桌面上的木块

分析过程：

  1. 隔离物体：木块
  2. 主动力：重力 G（竖直向下）
  3. 被动力：支持力 N（垂直接触面向上），无摩擦力（无相对运动趋势）
  4. 检查：G（地球施加），N（桌面施加）✓
例子2：静止在粗糙斜面上的木块

分析过程：

  1. 隔离物体：木块
  2. 主动力：重力 G（永远竖直向下！）
  3. 被动力：
    • 支持力 N（垂直斜面向上）
    • 静摩擦力 f（沿斜面向上，阻止下滑趋势）
  4. 检查：G（地球），N（斜面），f（斜面）✓

特别提醒：重力方向永远是竖直向下，不要画成垂直斜面向下！

思考题

一个木块放在水平传送带上，传送带以恒定速度向右运动。当木块刚放上去时（初速度为0），它受到的摩擦力方向是？

查看答案

摩擦力方向向右。因为木块相对于传送带向左运动（木块速度小于传送带速度），所以受到向右的滑动摩擦力，使木块加速。

1.3 挑战性例题：破除思维定势

例子3：支持力不等于重力的情况

一个质量为m的物体放在水平地面上，用一个斜向下的力F（与水平方向成θ角）压住物体，物体保持静止。

关键认识：支持力 N ≠ mg！

竖直方向受力平衡：N = mg + Fsinθ

支持力大于重力，因为外力F有向下的分量。

例子4：静摩擦力方向的判断

一个物体靠在竖直墙壁上，用水平力F压住它使其静止。分析物体的受力情况。

分析要点：

  • 水平方向：N = F（墙对物体的支持力等于推力）
  • 竖直方向：f = mg（摩擦力竖直向上，平衡重力）
  • 思维突破：摩擦力不一定在水平方向！这里摩擦力是竖直向上的。
综合思考题

传送带以速度v向右匀速运动，一个木块以速度2v向右滑上传送带。问：

  1. 木块刚滑上传送带时，受到的摩擦力方向？
  2. 当木块速度减小到v时，摩擦力如何？
  3. 如果传送带足够长，最终木块的运动状态？
查看答案
  1. 摩擦力向左。木块相对传送带向右运动（2v-v=v），受到向左的滑动摩擦力。
  2. 摩擦力变为0。木块与传送带速度相同，无相对运动，无摩擦力。
  3. 木块与传送带一起以速度v向右匀速运动，无摩擦力。
小结：受力分析口诀

一隔离，二重力
三找接触判弹力
四看运动判摩擦
五查施力不臆造

建模思想

受力分析本身就是一个物理建模的过程。我们将实际的物体简化为质点，将复杂的表面相互作用简化为明确的弹力和摩擦力。这种简化让我们能用数学工具来描述和预测物理现象。

第二章：核心分析方法论（武器篇）

目标

熟练掌握处理复杂系统和多力问题的两大核心方法：正交分解法和整体法与隔离法。

2.1 正交分解法——"化整为零"的艺术

核心思想

将一个矢量（通常是力或加速度）分解到两个相互垂直的坐标轴上，把复杂的二维平面问题转化为两个独立的一维直线问题。

x轴方向：F合x = max
y轴方向：F合y = may

如何选择坐标轴？【核心技巧】

坐标轴选择的黄金原则
原则 适用情况 优势
原则一（首选）
沿加速度方向建轴		 物体有明确的加速度方向 一个方向变为平衡方程
F合⊥ = 0
原则二（备选）
让更多力落在轴上		 静止或匀速直线运动
（a = 0）		 减少力的分解
降低计算复杂度

口诀：有加速度，就沿加速度方向建轴；无加速度，就让尽量多的力免于分解。

典型例题：斜面上的加速物块

题目：质量m = 2kg的物块，在与水平方向成θ = 37°角的恒力F = 20N作用下，沿粗糙水平地面向右做匀加速直线运动。物块与地面间的动摩擦因数μ = 0.5。求物块的加速度a和地面对物块的支持力N。

（g = 10 m/s²，sin37° = 0.6，cos37° = 0.8）

解题步骤：
  1. 受力分析
    • 重力 G = mg = 20N（竖直向下）
    • 支持力 N（竖直向上）
    • 拉力 F = 20N（与水平成37°角）
    • 滑动摩擦力 f = μN（水平向左）
  2. 选择坐标轴

    加速度方向明确：水平向右。根据原则一，建立水平x轴（向右为正）和竖直y轴（向上为正）。

  3. 正交分解
    • Fx = Fcos37° = 20 × 0.8 = 16N
    • Fy = Fsin37° = 20 × 0.6 = 12N
  4. 列方程求解
    y方向（平衡）：N + Fy - mg = 0
    N + 12 - 20 = 0
    N = 8N
    x方向（加速）：Fx - f = ma
    其中 f = μN = 0.5 × 8 = 4N
    16 - 4 = 2a
    a = 6 m/s²

2.2 整体法与隔离法——"宏观与微观"的切换

方法对比
方法 核心思想 优点 缺点
整体法 将系统看作一个整体
忽略内力		 求系统加速度最快
方程简单		 无法求内力
隔离法 分析单个物体
考虑所有作用力		 可以求内力 方程较多
计算复杂
使用策略【黄金法则】
  1. 判断能否用整体法

    唯一条件：系统内所有物体的加速度大小和方向必须完全相同

  2. 根据所求选择方法
    • 只求系统加速度 → 优先用整体法
    • 要求内力（拉力、压力等） → 必须用隔离法
    • 既求加速度又求内力 → 先整体后隔离
典型例题：连接体问题

题目：光滑水平面上，质量mA = 2kg和mB = 3kg的两物体用轻绳连接。用水平力F = 10N拉物体B，求：

  1. 系统的加速度a
  2. 绳子的拉力T
解题过程：

(1) 求系统加速度a——使用整体法

  • A和B加速度相同，可用整体法
  • 整体质量：M = mA + mB = 5kg
  • 整体受力：只有外力F = 10N
  • 由F = Ma得：a = F/M = 10/5 = 2 m/s²

(2) 求绳子拉力T——使用隔离法

隔离物体A（受力较简单）：

  • A只受绳子拉力T（水平向右）
  • 对A应用牛顿第二定律：T = mA·a
  • T = 2 × 2 = 4N

验证：隔离物体B

  • B受拉力F（向右）和绳子拉力T（向左）
  • F - T = mB·a
  • 10 - T = 3 × 2
  • T = 4N ✓
思考题

三个相同的物块A、B、C叠放在光滑水平面上，用水平力F拉最下面的C。问：

  1. 能否用整体法求系统加速度？
  2. 如何求A和B之间的摩擦力？
  3. 如果C与地面有摩擦，分析方法有何改变？
查看提示

1. 可以，因为三个物体一起运动，加速度相同

2. 隔离A，A受到的摩擦力提供其加速度

3. 仍可用整体法，但要考虑地面摩擦力这个外力

第三章：中难题核心模型突破（模型篇）

目标

将繁杂的问题归纳为有限的物理模型，学会识别并套用。重点掌握临界与极值问题、瞬时问题的分析方法。

3.1 临界与极值问题——"恰好"状态的数学表达

核心思想

临界问题的本质是分析一个物理过程的转折点。当某个物理量变化时，物体的运动状态也随之变化。临界状态就是这两种状态的分界点。

关键词识别："刚好"、"恰好"、"至多"、"至少"、"最大"、"最小"

临界条件总结

临界状态 临界条件 数学表达
即将滑动 静摩擦力达到最大值 fs = μsN
即将脱离接触 接触面弹力为零 N = 0
绳子即将断裂 拉力达到最大值 T = Tmax
圆周运动最高点 仅重力提供向心力 mg = mv²/r
例题1：斜面上物体的临界滑动

题目：质量m = 2kg的物体放在倾角θ = 37°的斜面上，静摩擦因数μs = 0.5。用平行于斜面向上的力F拉物体，求：

  1. 使物体保持静止，F的取值范围
  2. 若F = 25N，物体的加速度（动摩擦因数μk = 0.3）

（g = 10m/s²，sin37° = 0.6，cos37° = 0.8）

向上 f向下 37°
解答：

(1) 静止时F的范围

  • 垂直斜面：N = mgcosθ = 2×10×0.8 = 16N
  • 最大静摩擦力：fmax = μsN = 0.5×16 = 8N
  • 重力沿斜面分量：mgsinθ = 2×10×0.6 = 12N

临界状态一：即将下滑（F最小）

摩擦力沿斜面向上，达到最大值

Fmin + fmax = mgsinθ
Fmin + 8 = 12
Fmin = 4N

临界状态二：即将上滑（F最大）

摩擦力沿斜面向下，达到最大值

Fmax = mgsinθ + fmax
Fmax = 12 + 8
Fmax = 20N

答案：4N ≤ F ≤ 20N

(2) F = 25N时的加速度

F > Fmax，物体向上滑动，受滑动摩擦力

fk = μkN = 0.3×16 = 4.8N
F - mgsinθ - fk = ma
25 - 12 - 4.8 = 2a
a = 4.1 m/s²（沿斜面向上）

3.2 瞬时问题——抓住"不变"与"突变"

瞬时问题的黄金法则
力的类型 瞬时变化特性 原因
弹簧弹力 不突变 形变量不能瞬时改变
绳子拉力 可突变 约束条件改变
接触面弹力 可突变 接触状态改变
摩擦力 可突变 随正压力突变
解题三步曲
  1. 分析变化前：求出各力大小，特别是弹簧弹力
  2. 分析变化后瞬间：哪些力消失/突变，哪些力保持不变
  3. 应用牛顿第二定律：求瞬时加速度
例题2：竖直悬挂系统的瞬时问题

题目：两个质量分别为mA = 1kg和mB = 2kg的物块，用轻质弹簧连接后用细绳悬挂，系统静止。剪断细绳瞬间，求A、B的加速度。

解答：

第一步：分析剪断前（平衡态）

  • 对B：F弹簧 = mBg = 2×10 = 20N（向上）
  • 对整体：T = (mA + mB)g = 3×10 = 30N

第二步：剪断瞬间的受力变化

  • 绳子拉力T：30N → 0（突变）
  • 弹簧弹力F弹簧：20N（保持不变）

第三步：求瞬时加速度

对A：F合A = mAg + F弹簧 = 10 + 20 = 30N（向下）
aA = F合A/mA = 30/1 = 30 m/s²（向下）

对B：F合B = mBg - F弹簧 = 20 - 20 = 0
aB = 0
对比：剪断弹簧的情况

若剪断的是弹簧而非细绳，结果完全不同：

  • 弹簧被剪断，弹力立即消失（弹簧不存在了）
  • A：仍被绳拉住，aA = 0
  • B：自由落体，aB = g = 10 m/s²
思考题

如图，小车在光滑水平面上，车厢内用细绳和弹簧悬挂两个小球。剪断细绳瞬间：

  1. 左边小球的加速度是多少？
  2. 右边小球的加速度是多少？
  3. 小车会向哪边加速？
查看答案
  1. 绳被剪断，拉力消失，左球只受重力，a = g（向下）
  2. 弹簧弹力不突变，若原来平衡，则a = 0
  3. 系统水平方向动量守恒，左球向下运动，小车不会水平加速
瞬时问题分析流程

剪断前平衡分析
↓
弹簧弹力 = ?（记住此值）
↓
剪断后受力分析
↓
弹簧弹力不变 + 其他力突变
↓
F = ma 求加速度

第四章：弹簧弹力专题深度解析（专题突破）

目标

彻底攻克所有与弹簧相关的难题，建立对弹簧模型的深刻理解。从基本特性到复杂应用，从静力学到动力学，全方位掌握弹簧问题。

4.1 弹簧的核心特性再深化

胡克定律的本质理解
F = kx
  • F：弹簧的弹力（N）
  • k：弹簧的劲度系数（N/m），反映弹簧的"软硬"程度
  • x：弹簧的形变量（m），不是弹簧长度！

记住：x = |L - L₀|，其中L₀是弹簧原长

弹簧的瞬时特性（再次强调）

弹簧两端连接物体的加速度可以突变
但弹簧自身的弹力（由形变量决定）不能突变

4.2 弹簧的串并联

串联弹簧系统
eq F
串联特点：
  • 受力特点：各弹簧受力相等，F₁ = F₂ = F
  • 形变特点：总形变量等于各弹簧形变量之和，xtotal = x₁ + x₂
等效劲度系数推导：
由 F = k₁x₁ 得：x₁ = F/k₁
由 F = k₂x₂ 得：x₂ = F/k₂
总形变：xtotal = x₁ + x₂ = F/k₁ + F/k₂ = F(1/k₁ + 1/k₂)
等效弹簧：F = keq·xtotal
因此：1/keq = 1/k₁ + 1/k₂

结论：串联后系统变"软"，keq < min(k₁, k₂)

并联弹簧系统
eq
并联特点：
  • 形变特点：各弹簧形变量相等，x₁ = x₂ = x
  • 受力特点：总拉力等于各弹簧弹力之和，Ftotal = F₁ + F₂
等效劲度系数推导：
F₁ = k₁x，F₂ = k₂x
总拉力：Ftotal = F₁ + F₂ = k₁x + k₂x = (k₁ + k₂)x
等效弹簧：Ftotal = keq·x
因此：keq = k₁ + k₂

结论：并联后系统变"硬"，keq > max(k₁, k₂)

4.3 弹簧振子模型

简谐运动的条件

回复力F与位移x成正比，且方向相反：F = -kx

加速度：a = F/m = -(k/m)x

这是简谐运动的动力学特征

水平弹簧振子
周期公式：
T = 2π√(m/k)

注意：周期只与m和k有关，与振幅A无关！

4.4 弹簧与复杂情景的组合

例题：弹簧-斜面系统

质量m = 1kg的物块通过劲度系数k = 100N/m的弹簧连接在光滑斜面底部。斜面倾角θ = 30°。将物块从弹簧原长位置静止释放，求：

  1. 物块的平衡位置
  2. 物块的最大速度
  3. 振动的振幅
解答：

(1) 平衡位置

在平衡位置，弹簧弹力等于重力沿斜面的分量：

kx₀ = mgsinθ
x₀ = mgsinθ/k = 1×10×0.5/100 = 0.05m = 5cm

(2) 最大速度

最大速度出现在平衡位置。从原长到平衡位置，由能量守恒：

mgx₀sinθ = ½kx₀² + ½mvmax²
vmax = x₀√(g·sinθ/x₀ - k/2m)
vmax = 0.05√(100 - 50) = 0.35 m/s

(3) 振幅

从原长释放，振幅A = x₀ = 5cm

挑战题

两个相同的弹簧（k = 200N/m）和两个相同的物块（m = 0.5kg）按下图连接。整个系统竖直悬挂，求：

  1. 两弹簧的伸长量
  2. 剪断连接两物块的绳子瞬间，各物块的加速度
查看答案

(1) 伸长量：

  • 下弹簧：只承受B的重力，Δx₂ = mg/k = 0.5×10/200 = 0.025m = 2.5cm
  • 上弹簧：承受A和B的总重力，Δx₁ = 2mg/k = 2×0.5×10/200 = 0.05m = 5cm

(2) 剪断瞬间：

  • 弹簧弹力不突变！
  • A：F = mg + kΔx₂ = 5 + 5 = 10N，aA = 20m/s²（向下）
  • B：F = mg - kΔx₂ = 5 - 5 = 0，aB = 0

5.1 动态平衡问题

动态平衡的特征

核心：物体始终处于平衡状态（F合 = 0），但某些力的大小或方向在缓慢变化

关键：过程是动态的，但每一瞬间都满足平衡条件

分析方法

方法一：解析法（正交分解）

最通用、最稳妥的方法，适用于三个或更多力的情况。

  1. 建立坐标系
  2. 正交分解各力
  3. 列平衡方程：ΣFx = 0，ΣFy = 0
  4. 分析变化过程中各力的关系
方法二：矢量三角形法

当物体受三个共点力平衡时，这三个力构成封闭的矢量三角形。

AC θ

分析：当C点向右移动时，θ增大

F = G·tanθ（θ增大，F增大）
FAC = G/cosθ（θ增大，FAC增大）

5.2 含传送带的受力分析

传送带问题三步法
  1. 判断相对运动：比较v物与v带
  2. 确定摩擦力方向：与相对运动（趋势）方向相反
  3. 分段分析：找出速度相等的时刻作为分界点
水平传送带典型过程
带 → 阶段1：v物 < v带 f v=0 阶段2：v物 = v带 v=v带 f = 0
倾斜传送带分析

关键：判断μmgcosθ与mgsinθ的大小关系

条件 运动情况 摩擦力
μkcosθ > sinθ 能达到共速 先滑动后静摩擦
μkcosθ < sinθ 不能达到共速 始终滑动摩擦

5.3 图像信息题

图像分析"三看"原则
  1. 看坐标轴：明确物理量和单位
  2. 看特殊点：起点、交点、转折点
  3. 看斜率和面积：理解物理意义
常见图像的物理意义
图像类型 斜率含义 面积含义
v-t图 加速度a 位移x
a-t图 加速度变化率 速度变化量Δv
F-x图 力的变化率 功W
F-t图 力的变化率 冲量I
例题：由v-t图推导受力

分析过程：

  1. 0-2s：a = 2.5 m/s²，F合 = ma（向前）
  2. 2-4s：a = 0，F合 = 0（平衡）
  3. 4-6s：a = -2.5 m/s²，F合 = ma（向后）

5.4 解题思想总结与检查清单

程序化解题流程
  1. 明确研究对象（单个物体或系统）
  2. 受力分析（按照科学分类方法）
  3. 画受力图（规范、完整）
  4. 建立坐标系（优化选择）
  5. 列方程（F合 = 0 或 F合 = ma）
  6. 求解验证（检查物理意义）
模型化思考

拿到题目先判断属于哪个模型：

  • 临界问题？→ 找临界条件
  • 瞬时问题？→ 弹簧弹力不突变
  • 连接体？→ 整体法+隔离法
  • 传送带？→ 判断相对运动
  • 动态平衡？→ 矢量三角形或解析法
综合挑战题

如图，质量m = 2kg的物块通过轻质弹簧连接在倾角θ = 30°的光滑斜面底部。现用平行斜面向上的力F缓慢拉动物块，使弹簧从压缩状态变为拉伸状态。已知弹簧劲度系数k = 200N/m。

  1. 画出F-x图像（x为物块位移）
  2. 求F的最小值
  3. 当F = 15N时，撤去F，求物块的瞬时加速度
查看解题思路
  1. 建立坐标：以弹簧原长位置为原点
  2. 受力平衡：F + kx = mgsinθ（x为代数值，压缩为负）
  3. F = mgsinθ - kx，这是一条斜率为-k的直线
  4. Fmin = 0（当弹簧压缩量最大时）
  5. 撤去F瞬间，弹簧弹力不变，用F合 = ma求解
极限思维训练

对于不确定的情况，推到极端情况去思考：

  • 摩擦角：当倾角达到摩擦角时，物体恰好能静止
  • 拉力无穷大：弹力、摩擦力会如何变化？
  • 质量趋于零：加速度会趋于无穷大吗？
