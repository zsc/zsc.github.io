核心概念

受力分析是根据物体的运动状态和周围环境，分析物体受到的所有力的过程。它是解决力学问题的第一步，也是最关键的一步。

✨

建模思想

把复杂的实际问题简化为物理模型，是物理学的核心方法。受力分析的本质就是建立力学模型。

1.1 受力分析的"金科玉律"

📋

黄金三步法

  1. 明确研究对象 - 确定要分析哪个物体
  2. 分析物体状态 - 静止？匀速？加速？
  3. 寻找施力物体 - 每个力都要找到"谁给的"

图1：斜面上物体的受力分析示例

1.2 力的分类体系

分类依据 类型 特点 常见例子
按性质分 重力 始终竖直向下 G = mg
弹力 垂直于接触面 压力、支持力、拉力
摩擦力 平行于接触面 静摩擦、滑动摩擦
场力 不需要接触 电场力、磁场力
按接触分 接触力 需要直接接触 弹力、摩擦力
非接触力 超距作用 重力、电磁力
⚠️

特别提醒

避免使用"主动力"、"被动力"这样不科学的分类。应该根据力的性质和产生条件来分类。

🤔 思考题：一个静止在水平桌面上的木块，受到哪些力？每个力的施力物体是什么？

答案：

  1. 重力 G：施力物体是地球，方向竖直向下
  2. 支持力 N：施力物体是桌面，方向竖直向上

因为木块静止，所以 G = N，二力平衡。

1.3 画受力图的完整步骤

📐

标准作图流程

  1. 画出研究对象（用简单几何图形表示）
  2. 标出重力（从重心画起，竖直向下）
  3. 分析接触点，画出弹力（垂直接触面）
  4. 判断相对运动趋势，画出摩擦力（平行接触面）
  5. 检查其他场力（电场力、磁场力等）
  6. 标注力的符号（G、N、f、F等）
